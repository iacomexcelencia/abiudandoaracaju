# Ajudando Aju - Turismo em Aracaju

  Aplicativo de turismo para Aracaju, Sergipe, Brasil.
  Desenvolvido para a Prefeitura de Aracaju.

  ## Stack
  - React + TypeScript + Vite (frontend)
  - Express.js (backend)
  - PostgreSQL + Drizzle ORM (banco de dados)
  - Tailwind CSS + Shadcn/ui (UI)

  ## Funcionalidades
  - QR Codes para pontos turísticos
  - Passaporte digital gamificado
  - Suporte multilíngue (PT, EN, ES)
  - Dashboard de analytics
  - Painel de transparência pública
